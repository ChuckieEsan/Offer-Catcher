# 生成测验题目

## Overview

此 Skill 用于从题库中随机抽取题目，生成测验供用户练习。

## Instructions

### 1. 收集题目数据

使用 Qdrant 检索题目数据：
- 调用 qdrant 的 `scroll_all` 方法获取所有题目
- 根据用户指定的过滤条件进行筛选：
  - company（公司）
  - position（岗位）
  - question_type（题目类型）
- 如果用户要求只包含有答案的题目，进行过滤

### 2. 随机选取题目

使用 Python 的 `random.sample` 从符合条件的题目中随机选取指定数量（默认为 5）

### 3. 输出题目

以 Markdown 格式输出测验题目：
- 每道题显示题目内容、公司、岗位、类型
- 提示用户如何查看答案

## Example Output

```
📝 随机测验 (5 道题)

**1. 什么是 Python 装饰器？**
   公司: 字节跳动 | 岗位: 后端开发 | 类型: knowledge

**2. 请解释快速排序的时间复杂度。**
   公司: 阿里 | 岗位: 算法工程师 | 类型: algorithm

...

💡 回复对应数字查看答案，如「查看第1题答案」
```

## Constraints

- 如果没有符合条件的题目，返回提示信息
- 优先返回有答案的题目（便于用户练习）
- 题目数量默认 5 道，可由用户指定
