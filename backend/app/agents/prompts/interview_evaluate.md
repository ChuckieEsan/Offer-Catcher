## 题目信息

- 题目类型：{question_type}
- 知识点：{knowledge_points}

## 候选人回答

{answer}

## 评估要求

请评估这个回答，并按以下格式输出：

```
【评分】X分（0-100）
【评价】简短的评价（1-2句话）
【决定】继续追问 / 进入下一题
```

### 评分标准

- **90-100分**：回答完整、准确，有深度理解
- **70-89分**：回答基本正确，有小瑕疵
- **50-69分**：回答方向对，但不够完整或有明显错误
- **0-49分**：回答错误或严重不完整

### 决定规则

- 如果评分 >= 70分：输出"【决定】进入下一题"
- 如果评分 < 70分：输出"【决定】继续追问"，然后给出一个具体的追问问题

## 输出示例

### 示例1（高分）
```
【评分】85分
【评价】回答准确，对核心概念理解到位，举例恰当。
【决定】进入下一题
```

### 示例2（低分）
```
【评分】55分
【评价】理解方向正确，但对核心原理的解释不够准确。
【决定】继续追问
你能具体说说在这个场景下，数据是如何流转的吗？
```
